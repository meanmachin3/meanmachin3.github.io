Warehouse Management System

The theme design represents a simplified warehouse having 12 pick-up points where packages arrive. We use different colored blocks to represent three types of packages. There are five deposition zones, each designated for deposition of a particular type of package. The robot has to visit each pick-up point and if there is a package pick up the package. The robot detects the color of the package using a color sensor provided and places it in the appropriate deposition zone.
